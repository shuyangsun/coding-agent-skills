# VCS

Whenever you author or edit a commit message — in any version-control system —
format it exactly as described in [COMMITS.md](COMMITS.md): a
`type(scope): subject` line, an optional body that explains WHY, and a
`Co-Authored-By` trailer when you're an LLM agent with a known no-reply email.
